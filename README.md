# GraspSimulator
Brief description: This is a Unity-based grasping simulation project that includes scenes, models, materials, and example scripts. This README provides an overview of the project's purpose, how to run it, and the directory structure.
- Run: Use the executable under the Software folder, or open the Code directory with Unity to edit and run the project.
- Structure: Code/Assets contains resources; Packages/ and ProjectSettings contain Unity project configuration; Software contains built executables and runtime data.
