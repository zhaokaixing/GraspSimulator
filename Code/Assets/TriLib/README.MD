# TriLib

- 本目录包含 TriLib 相关库/资源（第三方库或运行时支持文件）。
- 说明与注意：

  - 若包含源码，优先保持 upstream 的目录结构并标注来源与版本号。
- 维护：
  - 更新库时先在独立分支测试，确认兼容性后合并主分支。
