# Scenes

简要（中文）
- 本目录包含 Unity 场景文件 (.unity) 和对应的 .meta 文件。
- 使用建议：
  - 场景命名为有意义的短句，例如 grasp_scene_main.unity。
  - 编辑场景请在 Unity 中进行，保存后提交 .unity 与 .meta。
  - 若场景要加入构建，请在 Unity 的 Build Settings 中添加并记录变更。
  - 避免在文本编辑器中直接编辑场景 YAML，可能导致损坏。

Brief (English)
- This folder stores Unity scene files (.unity) and their .meta.
- Guidelines: name scenes clearly, add scenes to Build Settings when needed, edit scenes in Unity Editor only.
